Test MGL PAX page

This is an experimental page. Here I'm embedding MGL-PAX documentation builder into a Coleslaw static blog generator.

This is accomplished by adding a Coleslaw extension which defines this method to render pages of "mgl-pax" type:

  • [method] COLESLAW:RENDER-TEXT TEXT (FORMAT (EQL MGL-PAX))

    Evaluates post's content as a Lisp file and interprets POST variable as MGL-PAX documentation section.

By the way, this post above was created using MGL-PAX's locative inserted into this blog post's body:

(coleslaw::render-text (method nil (t (eql :mgl-pax))))
